Frank's website style guide.

A continously updating styleguide

Welcome to my website style guide.

Funny: Learing to code is funny. When I started with this site, I had no action plan. I just started. At some point I decided that it would be easier if I coded things differently, making better use of class and implementing the use of ID. I started changing the HTML and CSS. After several hours of hard work and redesigning my buildup, I hit that refresh page: and everything looked exactly the same! This was exactly what was supposed to happen and I have never been more happy with seeing no change at all, because knowing that in the long run this will save me tons of work.

On this page you'll find various color sets that I'll use as basis for designing websites. Each set comes with a small introduction on how to apply the set. They are set up in similar order. The first color is the main background color. Second color is the color used for content boxes, while the third color is used to extra style header content. The forth color is mainly used to style the fonts and borders used. Aditional colors can be used in certain cases to make things more clear. Example is the white font color used in color set 1 for Dark Sienna so that the text can actually be read.
Updated the colors to use hsla formatting. Hsla allows the user to finetune colors by adjusting the saturation and ligt paramaters. This alows for a lot more control of the colors that other formats don't provide. Because of this, hsla is the preferred format used by designers.

Below the color sets you'll find examples of various types of fonts. I'll use some of these fonts on my websites, while others are just there for demonstration. The fonts are displayed with various text styles. It starts with plain, followed by strong, Italic and underline. Below that there are examples of each header style, starting with h1 all the way to h6. After that you'll see numers and symbols in the font used. The text that's used is the most used text to display fonts because it has all the alphabet letters in it, acording to Wikipedia.

I started this site as a challenge project from Codecadamy. As my skill progresses over time, so will this website. Most importantly it'll get more content, but also get more functionality and features. One of the features implented is the ability to change the style of this website. To do so, click on the link above the color set you want to apply and watch the site change before your eyes.

Additional content might be added at a later time. This can be any kind of content that can become in handy when designing and styling websites. For example button designs and their corresponding CSS.

Various colorsets as basis for websites.

Color set 1

Thistle

hsla(300, 24%, 80%, 1)

Tan

hsla(34, 44%, 69%, 1)

Steel blue

hsla(207, 44%, 49%, 1)

Dark sienna

hsla(354, 67%, 14%, 1)

Thistle is set as the body background color. Tan is used for box element background to differenciate from the body background. For the header background the color steel blue is used. This color can also be used as background for content boxes that you want to emphasize from others. Lastly dark sienna is used as the font color, border color and the likes of that.

Color set 2

Light green

hsla(120, 73%, 75%, 1)

Wild blue yonder

hsla(217, 37%, 68%, 1)

Cyber grape

hsla(267, 32%, 34%, 1)

Dark spring green

hsla(160, 93%, 23%, 1)

The body background color is Light green. Wild blue yonder is used as box element background including the headers. The emphasis on headers and similar elements come from the cyber grape font and bordercolor. Dark spring green is then used as the standard font and bordercolor.

Color set 3

White

hsla(0, 0%, 100%, 1)

Moccasin

hsla(36, 100%, 85%, 1)

Pale cornflower blue

hsla(210, 100%, 85%, 1)

Smalt

hsla(220, 100%, 85%, 1)

For this set, white is the main background color while mocassin is used as the content box background. To distinguish certain boxes from others, the color pale cornflower blue is used as background. Font and bordercolors are smalt in color
